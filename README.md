## Setup

Install dependencies :

```bash
pnpm install
```

### Before build and deploy

Run prettier, then run linter. To make sure there's no error or warning message :

```bash
pnpm run format; pnpm run lint
```

Run to build :

```bash
pnpm run build
```

Then deploy to github pages:

```bash
pnpm run deploy
```
